# 云笔记plus

云笔记plus项目是一个看板式的管理（笔记）项目，分为团队和个人用户两大模块，同时也配备后台管理功能。

## 安装

mysql数据库+redis服务器

请确保本地mysql数据库和redis服务器能正常运行

## API

```java
/board/{id}  //进入某个看板
```
